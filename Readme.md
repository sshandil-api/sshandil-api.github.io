 Web Development Assignment - README
    ====================================

    Project Overview
    ----------------
    This is a static web portfolio project demonstrating foundational web development skills 
    in HTML, CSS, and JavaScript. The site showcases a personal portfolio with projects, 
    contact information, and dark mode functionality.

    Features Implemented
    --------------------
    ✓ Semantic HTML5 structure with proper heading hierarchy
    ✓ Accessible design with ARIA labels and roles (role="region", aria-labelledby)
    ✓ Responsive meta viewport for mobile compatibility
    ✓ Navigation menu with anchor links to page sections
    ✓ Dark mode toggle button for user preference
    ✓ Projects section with images and figure/figcaption elements
    ✓ Contact form with required field validation
    ✓ Footer with copyright information
    ✓ External CSS and JavaScript file linking

    Project Structure
    -----------------
    Index.html       - Main HTML file with semantic markup
    styles.css       - External stylesheet for layout and design
    script.js        - JavaScript file for interactivity (dark mode, form handling)
    project1.jpg     - Screenshot for Project One (HTML)
    projectcss.jpg   - Screenshot for Project Two (CSS)
    project3.jpg     - Screenshot for Project Three (JavaScript)

    Sections
    --------
    1. Header - Page title and tagline
    2. Navigation - Links to Home, About, Projects, Contact
    3. Main Content - About section with brief introduction
    4. Projects - Three project cards demonstrating HTML, CSS, JavaScript skills
    5. Contact - Form for visitor inquiries
    6. Footer - Copyright notice

    Technologies Used
    ------------------
    • HTML5 - Page structure and semantic elements
    • CSS3 - Styling and layout (external stylesheet)
    • JavaScript - Dark mode toggle and form interactivity
    • Accessible design patterns - ARIA attributes and semantic markup

    How to Run
    ----------
    1. Open Index.html in a web browser
    2. Use VS Code Live Server for local development
    3. Toggle dark mode with the button in the header
    4. Fill out and submit the contact form

    Accessibility Features
    -----------------------
    • Semantic HTML structure (header, nav, main, section, footer)
    • Descriptive alt text for all images
    • Form labels properly associated with inputs
    • ARIA roles and labelledby attributes for screen readers
    • Clear heading hierarchy (h1 > h2 > h3)
